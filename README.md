# jiadun-dev-marketplace

A Claude Code plugin marketplace for sharing skills, plugins, agents and more.

## Install

```bash
/plugin marketplace add WangTingYeYe/jiadun_dev_marketplaces
```

## Browse & Install Plugins

```bash
/plugin marketplace browse jiadun-dev-marketplace
/plugin install <plugin-name>@jiadun-dev-marketplace
```
